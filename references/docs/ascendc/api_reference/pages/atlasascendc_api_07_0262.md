#### 功能说明

该接口废弃，并将在后续版本移除，请不要使用该接口。

计算给定输入张量和权重张量的2-D卷积，输出结果张量。Conv2d卷积层多用于图像识别，使用过滤器提取图像中的特征。

#### 约束说明

- 该接口当前不支持W=Kw并且H>Kh的场景，其将产生不可预期的结果。
- 操作数地址对齐要求请参见通用地址对齐约束。
