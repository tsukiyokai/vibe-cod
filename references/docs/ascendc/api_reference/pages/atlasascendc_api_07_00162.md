# RTC简介

RTC是Ascend C运行时编译库，通过aclrtc接口，在程序运行时，将中间代码动态编译成目标机器码，提升程序运行性能。

使用aclrtc接口前需要包含如下头文件：

| 1   | #include"acl/acl_rt_compile.h" |
| --- | ------------------------------ |
