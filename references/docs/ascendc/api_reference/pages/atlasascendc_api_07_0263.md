#### 产品支持情况

#### 功能说明

该接口废弃，并将在后续版本移除，请不要使用该接口。

根据输入的切分规则，将给定的两个输入张量做矩阵乘，输出至结果张量。将A和B两个输入矩阵乘法在一起，得到一个输出矩阵C。

- 操作数地址对齐要求请参见通用地址对齐约束。
