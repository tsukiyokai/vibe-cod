#### 功能说明

对标量的数据类型进行转换。

#### 函数原型

| 12  | template<typenameT,typenameU,RoundModeroundMode>__aicore__inlineUScalarCast(TvalueIn) |
| --- | ------------------------------------------------------------------------------------- |
